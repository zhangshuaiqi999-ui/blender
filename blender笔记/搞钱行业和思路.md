

# 搞钱组成

既然你决定深耕美妆（瓶瓶罐罐）领域作为赚钱的切入点，那么“专业性”和“效率”就是你的生命线。为了让你在 PC 修好后能直接进入“流水线式”生产，我将这两个维度的需求拆解为一份**全流程清单**。

你可以直接将这个清单同步到你的 GitHub 仓库中，作为你的**项目核对表（Checklist）**。

------

## 一、 技术构成：生产力的硬指标

这一部分决定了你的画面“真不真”、“贵不贵”。

### 1. 形状 (Modeling & Geometry)

- **高精度建模**：不仅仅是外壳，必须包含瓶壁厚度（Solidify）、内胆、吸管、泵头细节。
- **倒角逻辑 (Beveling)**：现实中没有绝对的直角。学会使用 `Bevel` 修改器和 `Weighted Normal` 提升光影转折的平滑度。
- **低模+高模 (Retopology/Subdiv)**：掌握细分建模逻辑，确保渲染时圆柱体没有棱角。

### 2. 材质 (Shading/Nodes)

- **PBR 基础流**：金属、塑料、橡胶的物理参数预设。
- **进阶玻璃节点**：包含 `Absorption Volume`（控制液体深浅颜色）、`Dispersion`（色散，模拟昂贵玻璃感）。
- **表面微瑕疵 (Surface Imperfections)**：积累指纹、灰尘、微划痕的黑白贴图，通过 `Noise Texture` 混合到 `Roughness` 和 `Normal` 频道。
- **标签系统 (Labeling)**：掌握 Alpha 通道贴图，实现丝印、烫金、凸凹文字效果。

### 3. 流体、液体与膏体 (Physics)

- **静态液体**：使用几何节点（Geometry Nodes）制作瓶身冷凝水珠（Droplets）。
- **动态流体**：小规模 `Flip Fluid` 或 `Mantaflow` 模拟精华液滴落、泵头喷雾。
- **膏体与粘稠物**：利用 `Shape Keys` 或 `Lattice` 模拟面霜被挑起时的形变。

### 4. 灯光与渲染 (Lighting & Rendering)

- **灯光阵列**：积累 3-5 套标准美妆布光（如：三点布光、环形柔光、背光勾勒边缘）。
- **反光板 (Light Cards)**：学会用发光面（Emission）作为反光板，手动精确控制瓶身的高光条（Highlights）。
- **渲染设置**：多层渲染（AOV/Render Passes），导出 `Depth`（深度）、`Crypto Matte`（选区）、`Glossy`（高光）方便后期微调。

------

## 二、 思路构成：商业感的软实力

这一部分决定了你的视频“像不像大片”、“甲方满不满意”。

### 1. 动画与运镜 (Animation & Cinematography)

- **关键帧曲线 (Graph Editor)**：告别匀速运动。掌握线性、贝塞尔曲线，做出“快慢快”的节奏感。
- **镜头焦段**：美妆多用 50mm - 100mm 以上的长焦镜头，消除透视畸变，增加高级感。
- **空镜与转场**：设计物体旋转、平移与相机推拉的配合，实现无缝剪辑点。

### 2. 环境与氛围 (Environment/Props)

- **场景资产**：丝绸（Cloth Sim）、大理石台面、自然光影（Gobo，模拟窗影/叶影）。
- **构图法则**：黄金分割、对角线构图，确保视觉重心永远在品牌 Logo 上。

### 3. 音乐与声音设计 (Sound Design/ASMR)

- **节奏同步 (Beat Mapping)**：剪辑点必须卡在音乐重音上。
- **音效素材 (SFX)**：
  - *质感类*：玻璃碰撞声、液体流动的粘稠声、拧盖子的阻尼声。
  - *氛围类*：高级的低音（Sub Bass）、转场音效（Whoosh）。

### 4. 剪辑与后期合成 (Editing & Compositing - DaVinci Resolve)

- **色彩空间管理**：理解 ACES 流程或 Blender 的 AgX 调色，确保颜色在各设备上一致。
- **后期特效**：光晕（Glow/Bloom）、胶片颗粒（Film Grain）、镜头畸变（Lens Distortion）。
- **重映射 (Speed Ramping)**：在达芬奇中二次调整动画速度，让节奏更加丝滑。

# 要把技术思维，收敛到这个细分领域中了。

这正是从“爱好者”转向“职业选手”的关键一步。**收敛技术思维**意味着你不再去研究所有炫酷的节点，而是只攻克那些能让“瓶瓶罐罐”看起来更贵、更真实的**特定技术路径**。

为了让你在 GitHub 笔记中建立起这套收敛后的知识体系，我为你把之前的清单细化成了美妆行业的**“底层逻辑操作指南”**。

------

### 1. 材质收敛：从“万能材质”到“极致质感”

在美妆领域，你只需要精通这四种材质的变体，就能应对 90% 的单子：

- **玻璃逻辑 (Advanced Glass)：**
  - **收敛点**：不要只调 `Transmission`。必须加入 `Volume Absorption`（体积吸收），让瓶子在转角厚度处呈现出深色，这才是“重工业”玻璃感。
  - **节点关键词**：`Light Path` (用于剔除阴影中的黑影), `Geometry (Pointiness)` (用于给瓶边缘加高亮)。
- **半透明塑料 (SSS Plastic)：**
  - **收敛点**：美妆盖子多用次表面散射（Subsurface Scattering）。
  - **技术指标**：模拟光线透入瓶盖边缘的微红或柔和感，这是廉价感与高级感的分水岭。
- **拉丝与电镀金属 (Anodized Metal)：**
  - **收敛点**：侧重各向异性（Anisotropic）反射。
  - **技术指标**：模拟泵头那种环形拉丝感，让高光呈放射状分布。

------

### 2. 灯光收敛：从“照亮物体”到“塑造价值”

美妆广告的灯光不是为了看清产品，而是为了制造“视觉舒适度”。

- **高光控制 (The Highlights)：**
  - **收敛点**：使用长条状的发光面（Area Light），配合 `Gradient Texture`（渐变贴图）。
  - **目的**：在瓶身拉出一条完美的长条高光，这在美妆界被称为“高光灵魂”。
- **边缘勾勒 (Rim Lighting)：**
  - **收敛点**：通过背光让透明瓶子的轮廓发光，使其从背景中剥离出来。
  - **技术逻辑**：利用 `Fresnel` 节点在后期或渲染时加强边缘。

------

### 3. 动画收敛：从“花里胡哨”到“优雅克制”

- **运镜收敛 (Cinematography)：**
  - **核心逻辑**：长焦（80mm+）+ 微距（Macro）。
  - **动作**：只练三种：极慢推近（Push-in）、环绕（Orbiting）、以及物体自转（Rotation）。
- **物理模拟收敛 (Focused Simulation)：**
  - **核心逻辑**：几何节点（Geo Nodes）水珠。
  - **实战**：不需要练大规模海啸，只练如何在瓶身上生成自然挂壁、大中小错落有致的水滴。

------

### 4. 后期收敛（达芬奇）：从“通用调色”到“肤感色调”

- **色彩倾向**：低饱和、高明度。
- **必练环节**：
  - **Glow & Mist (柔光与薄雾)**：给高光处加一点晕染，制造“仙气”。
  - **音效库积累**：只存玻璃清脆声、泵头按压声、丝绸拂过声。

------
