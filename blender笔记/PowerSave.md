# PowerSave简单使用说明

![image-20260119222158522](C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260119222158522.png)

这是一个非常有名的免费 Blender 插件，叫做 **PowerSave**。

它不是 Blender 原生的功能，而是一个第三方插件（通常是免费的）。它的核心作用是**“递增保存” (Incremental Save)** 和**“增强版自动保存”**，非常适合做项目时防止文件丢失或方便回溯旧版本。

以下是图中各个部分的详细解释：

### 1. 核心功能区 (顶部)

- **PowerSave (hops)**: 这是主保存按钮。点击它，插件会自动保存当前文件。
  - **最大亮点**：如果你勾选了设置，每次点击它，它可能会自动给文件名加数字后缀（例如从 `未命名_01` 变成 `未命名_02`），这样你就不会覆盖之前的进度，方便随时找回旧版本。
- **未命名_01**: 显示当前文件的名称。
- **左右箭头 (◀ ▶)**: 这是一个非常实用的功能，用于在**不同版本之间切换**。
  - 如果你保存了 `Project_01`, `Project_02`, `Project_03`，点击左箭头可以直接打开 `Project_02`，而不需要去文件夹里找。

### 2. 快捷工具 (中间)

- **Open Project Folder (打开项目文件夹)**: **这个功能完美解决了你上一个问题。** 点击它，会直接在 Windows 资源管理器中弹出当前 `.blend` 文件所在的文件夹，不用你自己去找文件存在哪了。

### 3. 自动保存设置 (Use Autosave)

- **Autosave Interval (2)**: 设置每 **2分钟** 自动保存一次。
- **Autosave Versions (0)**: 设置保留多少个自动保存的版本（0通常表示不限制或使用默认逻辑）。
- **Autosave Overwrite**: 选择自动保存是覆盖原文件还是创建新文件。

### 4. 清理与打包 (底部)

- **Toggle Autopack**: 自动打包。勾选后，你的贴图纹理会自动保存进 `.blend` 文件里（防止贴图丢失），但文件体积会变大。
- **Purge Orphans (清理孤立数据)**: 这是一个“大扫除”按钮。它会删除你场景里没用到的材质、网格等垃圾数据，帮文件“减肥”。

### 总结

你安装了这个插件是非常好的习惯！**PowerSave** 是许多专业 Blender 用户的标配，因为它能救命（防止崩溃丢失进度）。

**建议用法：**

养成习惯，不要按 `Ctrl+S`，而是点击这个面板上的大按钮（或者设置快捷键绑定到 PowerSave），这样你的项目就会自动变成 `模型_01`, `模型_02`... 一步步记录你的制作过程。
