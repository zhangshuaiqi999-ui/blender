这份 **《Typora 快捷键与功能速查手册 (Windows版)》** 是为你整理的。

由于 Typora 主打“所见即所得”，很多 Markdown 语法你甚至不需要背代码，直接用快捷键就能完成。建议你可以把这份文档收藏，或者复制到 Typora 里作为你的第一篇笔记练习。

------

### 1. 标题与段落 (最常用)

| **功能**                | **Windows 快捷键** | **Markdown 语法 (手动输入)** |
| ----------------------- | ------------------ | ---------------------------- |
| **一级标题**            | `Ctrl` + `1`       | `# ` + 文字                  |
| **二级标题**            | `Ctrl` + `2`       | `## ` + 文字                 |
| **三级标题**            | `Ctrl` + `3`       | `### ` + 文字                |
| ...以此类推             | `Ctrl` + `1` ~ `6` | `#` ~ `######`               |
| **清除标题 (变回正文)** | `Ctrl` + `0`       | (无)                         |
| **提升标题级别**        | `Ctrl` + "+"       | (如三级变二级)               |
| **降低标题级别**        | `Ctrl` + “-”       | (如二级变三级)               |

------

### 2. 文字样式 (强调与修饰)

| **功能**     | **Windows 快捷键**      | **Markdown 语法** | **备注**                     |
| ------------ | ----------------------- | ----------------- | ---------------------------- |
| **加粗**     | `Ctrl` + `B`            | `**文字**`        | Bold                         |
| **斜体**     | `Ctrl` + `I`            | `*文字*`          | Italic                       |
| **下划线**   | `Ctrl` + `U`            | `<u>文字</u>`     | 标准Markdown不支持，HTML标签 |
| **删除线**   | `Alt` + `Shift` + `5`   | `~~文字~~`        | 常用                         |
| **行内代码** | `Ctrl` + `Shift` + ```  | ``文字``          | 比如 `print()` 这种样式      |
| **注释**     | `Ctrl` + `-` (部分版本) | ``                | 不会在渲染后显示             |
| **高亮**     | (通常无默认，需设置)    | `==文字==`        | 需在偏好设置里开启 Highlight |

### 3. 列表与结构

| **功能**              | **Windows 快捷键**     | **Markdown 语法** |
| --------------------- | ---------------------- | ----------------- |
| **无序列表** (圆点)   | `Ctrl` + `Shift` + `]` | `- ` + 空格       |
| **有序列表** (数字)   | `Ctrl` + `Shift` + `[` | `1. ` + 空格      |
| **任务列表** (复选框) | `Ctrl` + `Shift` + `X` | `- [ ] ` + 空格   |
| **引用块**            | `Ctrl` + `Shift` + `Q` | `> ` + 空格       |
| **增加缩进** (下一级) | `Tab`                  | (在列表项中按)    |
| **减少缩进** (上一级) | `Shift` + `Tab`        | (在列表项中按)    |

### 5. 编辑与视图操作 (提升效率)

| **功能**       | **Windows 快捷键**     | **说明**                                         |
| -------------- | ---------------------- | ------------------------------------------------ |
| **查找**       | `Ctrl` + `F`           | 页面内搜索                                       |
| **替换**       | `Ctrl` + `H`           | 批量修改文字                                     |
| **源代码模式** | `Ctrl` + `/`           | **神器**。切换回“代码”样子，看背后的Markdown源码 |

### 💡 给你的特别建议

1. **一定要记住 `Ctrl + /` (源代码模式)：**

   虽然 Typora 是所见即所得，但有时候格式乱了（比如表格调整不灵了），按一下这个键切回“源码模式”手动改一下符号，是最快的解决办法。

2. **关于 `Alt + Shift + 5` (删除线)：**

   这个快捷键有点别手，如果你经常用删除线，可以在 Typora 的 `偏好设置` -> `通用` -> `高级设置` -> `conf.user.json` 里修改它（稍微有点极客，如果需要我可以教你）。

3. **数学公式和高亮：**

   如果你发现 `==高亮==` 或者 LaTeX 公式没反应，记得去 **偏好设置 (Preferences)** -> **Markdown** 面板里，把相关的扩展语法（Highlight, Diagrams 等）勾选上，重启 Typora 即可生效。
